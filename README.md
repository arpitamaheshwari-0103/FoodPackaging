# 📦 Food Packaging Analytics & Optimization Platform

## 🚀 Overview

This project is a data-driven analytics platform designed for the food packaging ecosystem in India.

It helps businesses such as:
- Cloud kitchens
- Restaurants
- FMCG / D2C brands
- Food manufacturers

to make smarter packaging decisions using data and machine learning.

---

## 🎯 Problem Statement

Food businesses face major challenges in packaging:
- High packaging costs
- Material wastage
- Supplier inefficiencies
- Delivery failures (damage/leakage)
- Lack of sustainability insights

This platform aims to solve these problems through data intelligence and predictive analytics.

---

## 🧠 Key Features

### 📊 Descriptive & Diagnostic Analytics
- Dataset exploration
- Feature importance visualization
- Cluster distribution analysis

---

### 🤖 Predictive Analytics

#### 🔹 Classification Model
- Predicts customer interest (Yes/Maybe/No)
- Metrics used:
  - Accuracy
  - Precision
  - Recall
  - F1 Score

---

#### 🔹 Regression Model
- Predicts:
  - Monthly packaging loss
  - Packaging spend

---

### 📊 Customer Segmentation
- K-Means clustering
- Identifies different customer groups:
  - Cost-sensitive businesses
  - Growth-focused brands
  - Mature enterprises

---

### 🔗 Association Rule Mining
- Uses Apriori algorithm
- Metrics:
  - Support
  - Confidence
  - Lift

---

### 📥 New Customer Prediction
- Upload new customer dataset
- Predict likelihood of using the platform
